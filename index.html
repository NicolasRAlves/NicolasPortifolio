Projetos

Inclusão de Presença - TOTVS Fluig

Processo desenvolvido na plataforma TOTVS Fluig, destinado ao controle de solicitações de presença em aulas e laboratórios. O processo permite registrar o solicitante, turma, alunos presentes/ausentes, anexos e observações. Interface responsiva com validações, facilitando a gestão acadêmica e administrativa.
Tecnologias: JavaScript, jQuery, HTML, CSS, TOTVS Fluig (ECM + BPM)

Solicitação de Pagamento - TOTVS Fluig

Processo criado em TOTVS Fluig para centralizar e automatizar solicitações de pagamento, com etapas de aprovação por gerência e diretoria. Inclui anexos de documentos fiscais, validação de dados financeiros e controle de rateios por centro de custo. Integração com sistemas externos como o Whinthor.
Tecnologias: JavaScript, jQuery, HTML, CSS, TOTVS Fluig, REST/SOAP APIs

Telas de Cadastro Internas - TOTVS Fluig

Desenvolvimento de telas dentro da plataforma TOTVS Fluig para cadastro de funcionários, projetos, clientes e atividades. O diferencial está na aplicação de princípios de UX/UI, criando uma interface mais moderna, responsiva e interativa para os usuários.
Tecnologias: HTML, CSS, JavaScript, jQuery, TOTVS Fluig, Bootstrap

API de Transações - Desafio Itaú

API REST para registrar transações e gerar estatísticas dos últimos 60 segundos, conforme desafio proposto pelo Itaú. Os dados são armazenados em memória, garantindo alta performance e simplicidade.
Tecnologias: Java, Spring Boot, Maven, JUnit, Insominia

JavaFarm - Simulador de Fazenda

Jogo educativo que simula o cultivo e a colheita de alimentos, com sistema de armazenamento no celeiro. Projeto voltado para o aprendizado de orientação a objetos, com aplicação de herança e polimorfismo.
Tecnologias: Java (POO), JavaFX, IntelliJ, Terminal

Montagem de Computador - Spring Boot

Sistema de montagem de PCs que permite ao usuário montar uma máquina personalizada verificando compatibilidade entre os componentes. Desenvolvido como desafio acadêmico.
Tecnologias: Java, Spring Boot, Thymeleaf, HTML, CSS, Bootstrap
